¿Que es un Chatbot?

Es una tecnologia de software que se comunica con el cliente ya sea como asistente o compañero virtual esto se puede integrar en algunas sistemas como Facebook, Twitter, Telegram, Whatsapp e incluso en la web

¿Para que sirve un Chatbot?

La principal funcion del Chatbot es llevar a cabo tareas o funciones de atencion al cliente, pedidos automaticos, informar sobre un producto.
Otra aplicacion seria de compañero virtual como para hacer preguntas o tener con quien conversar.

Ventajas del Chatbot

Ahorar en el costo de atencion al cliente

Despejar las principales dudas del usuario de manera rapida

Interaccion de forma facil con el cliente

Proporciona una experiencia agradable con el usuario

Desventajas del Chatbot

No reemplaza al 100% al ser humano

Las consultas suelen ser dificiles de predecir

Si se realiza con programcion funcional el codigo va a hacer muy grande y poco factible

Aplicaciones practicas de un Chatbot

Atencion al cliente

Comercio online

Compañero virtual
